\begin{DoxyItemize}
\item Header-\/only (optional!)
\item Supports the latest \href{https://toml.io/}{\texttt{ TOML}} release (\href{https://toml.io/en/v1.0.0}{\texttt{ v1.\+0.\+0}}), plus optional support for some unreleased TOML features
\item Passes all tests in the \href{https://github.com/BurntSushi/toml-test}{\texttt{ toml-\/test}} suite
\item Supports serializing to JSON and YAML
\item Proper UTF-\/8 handling (incl. BOM)
\item C++17 (plus some C++20 features where available, e.\+g. experimental support for \href{https://en.cppreference.com/w/cpp/keyword/char8_t}{\texttt{ char8\+\_\+t}} strings)
\item Doesn\textquotesingle{}t require RTTI
\item Works with or without exceptions
\item Tested on Clang (6+), GCC (7+) and MSVC (VS2019)
\item Tested on x64, x86 and ARM
\end{DoxyItemize}
