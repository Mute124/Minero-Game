\hypertarget{index_mainpage-example-speed-up-compilation}{}\doxysubsection{\texorpdfstring{Speeding up compilation}{Speeding up compilation}}\label{index_mainpage-example-speed-up-compilation}
Because toml++ is a header-\/only library of nontrivial size you might find that compilation times noticeably increase after you add it to your project, especially if you add the library\textquotesingle{}s header somewhere that\textquotesingle{}s visible from a large number of translation units. You can counter this by disabling header-\/only mode and explicitly controlling where the library\textquotesingle{}s implementation is compiled.

{\bfseries{Step 1\+: Set \doxylink{group__configuration_ga8219bfd7e75587516385b6022b28abc2}{TOML\+\_\+\+HEADER\+\_\+\+ONLY} to \mbox{[}code\mbox{]}0\mbox{[}/code\mbox{]} before including toml++}}

This must be the same everywhere, so either set it as a global {\ttfamily \#define} in your build system, or do it manually before including toml++ in some global header that\textquotesingle{}s used everywhere in your project\+: \textbackslash{}cpp global\+\_\+header\+\_\+that\+\_\+includes\+\_\+toml++.h

\doxylink{_classes_2lib_2include_2avx512fintrin_8h_a1b7950c88593de42a92cc8558f5488f6}{define} TOML\+\_\+\+HEADER\+\_\+\+ONLY 0 \#include \texorpdfstring{$<$}{<}\doxylink{toml_8hpp}{toml.\+hpp}\texorpdfstring{$>$}{>} \textbackslash{}endcpp

{\bfseries{Step 2\+: Define \doxylink{group__configuration_ga27ee69835a1ddff9dbaf3015b4b08a11}{TOML\+\_\+\+IMPLEMENTATION} before including toml++ in one specific translation unit}}

\textbackslash{}cpp some\+\_\+code\+\_\+file.\+cpp

{\bfseries{Bonus Step\+: Disable any library features you don\textquotesingle{}t need}}

Some library features can be disabled wholesale so you can avoid paying their the compilation cost if you don\textquotesingle{}t need them. For example, if all you need to do is serialize some code-\/generated TOML and don\textquotesingle{}t actually need the parser at all you, can set \doxylink{group__configuration_ga121912d75d4624de6ab6854e41d8a852}{TOML\+\_\+\+ENABLE\+\_\+\+PARSER} to {\ttfamily 0} to disable the parser altogether. This can yield fairly significant compilation speedups since the parser accounts for a good chunk of the library\textquotesingle{}s code.
