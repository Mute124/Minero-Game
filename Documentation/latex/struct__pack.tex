\doxysubsection{Detailed Description}
M3D file format structure 3DMO m3dchunk\+\_\+t file header chunk, may followed by compressed data PRVW preview chunk (optional) HEAD m3dhdr\+\_\+t model header chunk n x m3dchunk\+\_\+t more chunks follow CMAP color map chunk (optional) TMAP texture map chunk (optional) VRTS vertex data chunk (optional if it\textquotesingle{}s a material library) BONE bind-\/pose skeleton, bone hierarchy chunk (optional) n x \doxylink{structm3db__t}{m3db\+\_\+t} contains propably more, but at least one bone n x \doxylink{structm3ds__t}{m3ds\+\_\+t} skin group records MTRL\texorpdfstring{$\ast$}{*} material chunk(s), can be more (optional) n x \doxylink{structm3dp__t}{m3dp\+\_\+t} each material contains propapbly more, but at least one property the properties are configurable with a static array, see m3d\+\_\+propertytypes n x m3dchunk\+\_\+t at least one, but maybe more face chunks PROC\texorpdfstring{$\ast$}{*} procedural face, or MESH\texorpdfstring{$\ast$}{*} triangle mesh (vertex index list) or VOXT, VOXD\texorpdfstring{$\ast$}{*} voxel image (converted to mesh) or SHPE\texorpdfstring{$\ast$}{*} mathematical shapes like parameterized surfaces LBLS\texorpdfstring{$\ast$}{*} annotation label chunks, can be more (optional) ACTN\texorpdfstring{$\ast$}{*} action chunk(s), animation-\/pose skeletons, can be more (optional) n x \doxylink{structm3dfr__t}{m3dfr\+\_\+t} each action contains probably more, but at least one frame n x \doxylink{structm3dtr__t}{m3dtr\+\_\+t} each frame contains probably more, but at least one transformation ASET\texorpdfstring{$\ast$}{*} inlined asset chunk(s), can be more (optional) OMD3 end chunk
