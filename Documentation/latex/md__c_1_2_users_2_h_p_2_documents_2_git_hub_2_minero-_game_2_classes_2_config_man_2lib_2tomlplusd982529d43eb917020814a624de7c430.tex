 Contributions are very welcome! Either by \href{https://github.com/marzer/tomlplusplus/issues}{\texttt{ reporting issues}} or submitting pull requests. If you wish to submit a PR, please be aware that\+:


\begin{DoxyItemize}
\item The single-\/header file {\ttfamily \doxylink{toml_8hpp}{toml.\+hpp}} is generated by a script; make your changes in the files in {\ttfamily include}, {\bfseries{not}} in {\ttfamily \doxylink{toml_8hpp}{toml.\+hpp}}.
\item Your changes should compile warning-\/free on at least one of\+:
\begin{DoxyItemize}
\item GCC 8 or higher
\item Clang 8 or higher
\item MSVC 19.\+2X (Visual Studio 2019) or higher
\end{DoxyItemize}
\item You should regenerate the single-\/header file as part of your PR (a CI check will fail if you don\textquotesingle{}t).
\end{DoxyItemize}
