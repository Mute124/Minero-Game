\doxysubsection{Detailed Description}
A TOML path. 

\textbackslash{}detail This type parses and represents a path to a TOML node. It validates the syntax of the path but does not ensure that the path refers to a valid node in any particular TOML document. If parsing fails, the object will evaluate as \textquotesingle{}falsy\textquotesingle{}, and will be empty.
