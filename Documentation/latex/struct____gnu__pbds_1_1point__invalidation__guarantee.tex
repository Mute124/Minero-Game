\doxysubsection{Detailed Description}
Signifies an invalidation guarantee that includes all those of its base, and additionally, that any point-\/type iterator, pointer, or reference to a container object\textquotesingle{}s mapped value type is valid as long as its corresponding entry has not be erased, regardless of modifications to the container object.
