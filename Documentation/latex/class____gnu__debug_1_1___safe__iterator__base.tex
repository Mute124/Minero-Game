\doxysubsection{Detailed Description}
Basic functionality for a {\itshape safe} iterator. 

The \+\_\+\+Safe\+\_\+iterator\+\_\+base base class implements the functionality of a safe iterator that is not specific to a particular iterator type. It contains a pointer back to the sequence it references along with iterator version information and pointers to form a doubly-\/linked list of iterators referenced by the container.

This class must not perform any operations that can throw an exception, or the exception guarantees of derived iterators will be broken.
